About StegoCloud

An innovative and secure document storage platform built on the MERN stack.

How It Works

A user uploads a document, which is immediately encrypted using a strong algorithm.

A unique, random key is generated for the document. This key is then embedded within an image using steganography.

The encrypted document is stored in our database, but the key is not. The key is only accessible via the stego-image.

The user shares the document link with a recipient and sends the stego-image separately through a secure channel.

The recipient uploads the stego-image to the platform to extract the key, which then decrypts and grants access to the document.

Technology Stack

MongoDB

A NoSQL database for storing encrypted documents.

Express.js

A web framework for Node.js that handles our backend logic.

React.js

A JavaScript library for building the user interface.

Node.js

A JavaScript runtime that powers our server-side application.
